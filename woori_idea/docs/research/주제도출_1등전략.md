## A. 소비자가 "지금" 진짜 화내는 포인트 (현장 목소리 발췌)

> 카페·디씨·뉴스 댓글·고객의 말씀에서 반복적으로 등장하는 패턴

1. **"엄마가 또 보이스피싱 당했어요"** — 가족 목소리 클로닝, 검찰 사칭, 원격제어 앱. 막을 시점에 가족이 옆에 없음.
2. **"앱에서 ELS 사라고 권유받았는데 위험 설명을 영상 1분 30초 빨리 넘기기로 끝냈다"** — 디지털 채널 설명의무 형식화.
3. **"전화상담 30분 기다리다 끊었다"** — 민원 폭주 후처리, 콜센터 한계.
4. **"창구 직원이 정신없이 빨리 설명하고 사인하라더라"** — 영업점 설명의무 시간 부족.
5. **"앱 글씨가 너무 작고, OTP가 어디 있는지도 모르겠다"** (시니어/장애인) — 디지털 접근성.
6. **"리딩방·코인 사기당했는데 은행은 출금 막지 않더라"** — 청년 신종 사기.
7. **"민원 넣었는데 또 똑같은 답변만 돌아왔다"** — 민원 분류·응답 자동화의 그림자.

## B. 응모분야별 주제 후보 (각 분야 2~3개)

### 분야 1. 취약계층 권익보호
- **A1. 보호자-피보호자 연결형 "효(孝)지킴이" 계좌** — 시니어 부모 계좌 이상거래 시 자녀 앱에 실시간 알림 + AI 음성 통화로 "지금 받은 전화 진짜인지" 본인 확인 (보이스피싱 무과실 배상제 대비 사전 차단 핵심)
- **A2. WON 뱅킹 "쉬운모드" 멀티모달** — 시니어/저시력/지체장애 자동 감지(터치 속도·실수율·음성) → UI를 즉시 단순화 + 수어·음성·고대비 동시 출력 (온디바이스 LLM 3B 활용)
- **A3. 다문화/외국인 노동자용 17개 언어 실시간 통역 상담** — 영업점 태블릿에 동시통역+상품설명 RAG

### 분야 2. 불완전판매 예방
- **B1. "내가 이해했는지" AI 역질문 챗봇** — 상품 설명 후 AI가 소비자에게 핵심 리스크 4~5개를 본인 말로 답하게 요구. 정답 80% 미달 시 청약 진행 차단 → **금소법 설명의무 디지털 증빙**
- **B2. ELS·구조화상품 "최악시나리오 시뮬레이터"** — 본인 자산/연령/투자목적 입력 시 GenAI가 30초 영상으로 "당신이 이 상품에서 손실 본 미래 시점의 자기 자신"을 보여줌 (시각적 충격 = 효과성)
- **B3. 영업점 상담 녹취 실시간 컴플라이언스 AI** — 직원-고객 대화를 STT+LLM이 분석, 부당권유·설명누락 즉시 직원 화면에 경고 (텔러 코치)

### 분야 3. 민원 예방
- **C1. 민원 "전조 신호" 사전 예측 모델** — 콜센터 음성/앱 행동/거래 패턴에서 24~72시간 내 민원 발생 확률 예측 → 선제 콜백
- **C2. 우리원더(상담봇) 감정-격앙 감지 + 휴먼에스컬레이션** — LLM이 사용자 감정·이슈 복잡도 평가, 일정 임계치 초과 시 즉시 베테랑 상담사 연결
- **C3. 민원 답변 자동 초안 + "공감 점수" 평가기** — 상담사 답변을 LLM이 톤·공감도 평가 후 첨삭

### 분야 4. 금융사기(보이스피싱) 예방 ← **가장 핫한 분야**
- **D1. 온디바이스 LLM "통화 동석 AI"** — 사용자 통화 내내 AI가 동석해 보이스피싱/딥페이크 voice 실시간 탐지, 사기 의심 시 화면에 빨간 경고 + 30초간 송금 lock
- **D2. "가족 안심코드" — 보이스피싱 가족 클로닝 차단 인증** — 가족 간 사전 약속 단어/이미지/제스처를 우리은행 앱에 등록, 가족 사칭 송금 요청 시 AI가 발신자에 안심코드 요구
- **D3. AI 에이전트 결제 시대 대비 "에이전트 거래 본인의사 확인 레이어"** — 2026 금융결제원 PoC 추진 중인 AI 에이전트 결제에서 사용자 진의를 멀티팩터로 검증

### ★ 1순위 — "효(孝)지킴이 × 통화동석 AI" (보이스피싱+취약계층 결합)

**한 줄 설명:**
> 시니어 부모의 통화/송금 흐름을 **온디바이스 LLM**이 24시간 동석 감시하고, 위험 시점에 **자녀 앱**과 **우리은행 콜백 AI**가 동시에 끼어들어 송금 직전 멈춤.

**왜 1등인가:**
- **효과성(40)** — 무과실 배상제 시행 시 은행 부담 2,800억/년. **사전 차단율 1%만 올려도 약 28억 절감**. 정량 ROI 강력.
- **실현가능성(30)** — LG U+ ixi-O가 이미 온디바이스 LLM 보이스피싱 탐지 상용화. **WON뱅킹 앱 + 우리은행 AI콜봇 결합**으로 6개월 PoC 가능.
- **혁신성(30)** — "통화-앱-가족-은행 콜백" 4자 멀티에이전트 협업. CES 2026 발표 3~8B 온디바이스 모델 활용. 2026.5 가장 hot한 AI 에이전트 트렌드.
- **우리은행 정체성 결합** — 2026 "사전예방적 소비자보호" + "생성형 AI FDS" 두 슬로건을 한 화면에 시연.
- **감성 후크** — "엄마, 그 전화 받지 마세요"라는 자녀의 알림 메시지가 PT 슬라이드 첫 장에 박히면 심사위원 몰입 즉발.

**핵심 AI 활용 포인트:**
1. **온디바이스 LLM (3~8B)** — 통화 STT + 음성 클론(딥보이스) 탐지 + 사기 시나리오 분류
2. **멀티에이전트 협업** — 폰 AI ↔ WON뱅킹 ↔ 우리AI콜봇 ↔ 자녀 앱
3. **RAG** — 최신 사기 시나리오 DB(통합대응단·금감원 패턴) 실시간 업데이트
4. **연합학습(FL)** — 사용자 데이터 폰에 머무름, 프라이버시 보호

---

### ★ 2순위 — "내가 이해했는지 챗봇 + 최악시나리오 시뮬레이터"

**한 줄 설명:**
> 디지털·영업점에서 상품 청약 직전, AI가 소비자에게 **자기 말로 리스크 설명**을 요구하고, **30초 GenAI 영상**으로 "5년 뒤 손실 본 자기 모습"을 보여줘 충동 구매 차단.

**왜 강력한가:**
- 홍콩 ELS 사태(과징금 1.4조)의 직접 대응책. **우리은행은 ELS 5개에서 제외된 사실을 PT에서 부각** → "우리는 더 강력하게 한다"
- 금소법 6대 원칙 중 설명의무·적합성 원칙의 디지털 증빙. 법적 분쟁 시 우리은행 측 방어 자료
- **소라(Sora) 류 GenAI 영상 + LLM 챗봇** 결합 = 혁신성

**핵심 AI:**
- LLM 역질문 챗봇 (이해도 평가)
- GenAI 짧은 영상 생성 (개인화 손실 시뮬레이션)
- 적합성 평가 자동화 (KYC + 행동패턴)

## E. Q&A 예상 (심사위원 단골 질문)

- **"개인정보·통신비밀보호법 충돌 아닌가?"** → 온디바이스 처리 + 사용자 옵트인 + 가족 동의 절차
- **"오탐(false positive)으로 정상 통화 차단되면?"** → 2단계 알림(경고 → 송금만 lock), 가족 안심코드로 해제
- **"우리은행만 가능한 차별점?"** → 이사회 금소위 + 2026 사전예방 체계 + FDS 자산
- **"비용은?"** → 무과실 배상제 2,800억 vs 시스템 구축비 추정 50~100억 → ROI 명확
