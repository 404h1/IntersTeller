## 2. **1순위 — TalkBack AI** : "내가 이해했어요"를 AI가 검증하는 첫 은행

### 2-1. 한 줄 정의

> **금융상품 청약 직전, AI가 5개의 핵심 리스크를 "본인 말로 다시 설명해보세요"라고 역질문하고, LLM이 의미 일치도를 평가해 80% 미달 시 청약을 차단하는 시스템.** 부족한 부분은 GenAI가 30초짜리 "당신이 5년 뒤 손실 본 미래" 영상으로 재학습시킨다.

### 2-2. 왜 진짜 빈자리인가 (3중 검증)

**(1) 국내 9개 은행 — 누구도 안 함**
- 영업점 청약: 녹취·해피콜은 있지만 "이해 확인"은 직원의 질문 한두 마디. 형식적 이행 비판 지속.
- **디지털 청약(앱/홈페이지)**: 약관 동의 체크박스 한 번. 설명영상은 빨리감기 가능. 우리WON뱅킹·하나원큐·KB스타뱅킹 전부 동일.
- 우리은행 운영GPT, KB-GPT, 신한 자체 LLM 등은 **직원용 RAG**이거나 **상품 추천**용. **고객의 이해도를 평가하는 LLM은 없음.**
- 카뱅·NH의 XAI는 "왜 이 상품을 추천했는가" 설명. **"고객이 이해했는가" 검증은 없음.**

**(2) 정부 정책**
- 금소법 6대 원칙(설명의무·적합성·적정성 등) **이행 방식은 판매자 재량**. 디지털 채널 이해도 검증 의무화는 아직 없음 → **선점하면 표준 제시 가능**.
- 2026.1.1 금소법 시행령 개정으로 외화보험 적합성 원칙 추가 → 디지털 채널 설명의무 강화 필요성↑.

**(3) 글로벌·학술**
- 미국·EU에는 KYC/Suitability Test가 있으나 **사지선다 객관식 위주**. 자유 발화 이해도 평가는 학술 prototype(EMNLP 2024 "Comprehension Verification via LLM" 류) 단계.
- 일본 라쿠텐증권 2025 "AI 설명 챗봇"은 **고객 질문에 답하는 방향** → 역방향(고객이 이해했는지 평가)이 아님.

→ **9개 은행 ✗ / 정부 ✗ / 글로벌 ✗ — 진짜 빈자리.**

### 2-3. 왜 진짜 필요한가 — "감동"의 정량 근거

| 사실 | 출처 |
|---|---|
| 홍콩 ELS 5개 은행 과징금 **1조 4천억원** (2026.2) | [서울신문](https://www.seoul.co.kr/news/economy/finance/2026/02/12/20260212500476) |
| 분쟁조정 배상비율 30~65%, 피해자 수만 명 | 금감원 분쟁조정위원회 |
| **우리은행은 5개 명단에서 제외** — 보수적 판매로 사태 비껴감 | [01_banks_official.md], 한국금융신문 |
| 그러나 디지털 청약 비중은 매년 ↑ — 미래에도 ELS 같은 사태 차단할 수단 부재 | 우리WON뱅킹 비대면 청약 비중 60%+ |
| 위법계약해지권 행사 시 환불·소송 비용 발생 | 금소법 §47 |

**ROI 계산 (간단):**
- 시스템 구축비 추정: 30~50억원 (LLM 파인튜닝 + 청약 플로우 통합 + 6개월)
- 홍콩 ELS 한 건 평균 과징금/배상 → 우리은행 만약 1/5만 노출됐어도 약 2,800억 부담
- **차단율 1% 개선 = 약 28억 절감. 6개월 안에 본전.**

### 2-4. 와우 포인트 — PT에서 박수 나오는 5가지

1. **첫 슬라이드**: 실제 홍콩 ELS 피해자 인터뷰 음성 30초 → "직원이 빨리 사인하라 했어요. 위험은 따로 안 들었어요."
2. **두 번째 슬라이드**: "이런 일이 우리은행에선 일어나지 않았습니다. 다음에도 일어나지 않게 하겠습니다." (ELS 5개에서 제외된 사실 부각)
3. **데모**: 심사위원 한 명에게 태블릿 건네고 ELS 청약 시연. AI가 "이 상품에서 손실 날 수 있는 상황을 본인 말로 설명해주세요" → 심사위원이 "음… 주가가 떨어지면…" 답변 → AI가 "녹인 구간을 70% 이해하셨네요. 추가 영상 30초 보시겠어요?" → GenAI가 손실 시뮬레이션 영상 즉석 생성
4. **숫자**: "ELS 1.4조 부담을 28억으로 환산. 100배 ROI."
5. **마무리**: "다른 은행은 '왜 추천했는지' 설명합니다. 우리은행은 '당신이 이해했는지' 묻습니다."

### 2-5. 평가 기준 매핑

| 기준 | 점수 예상 | 근거 |
|---|---|---|
| **효과성(40)** | 38/40 | 정량 ROI 명확, 법적 분쟁 시 디지털 증빙으로 우리은행 방어, 위법계약해지권 행사율 감소 측정 가능 |
| **실현가능성(30)** | 27/30 | 우리WON뱅킹 청약 플로우 + 운영GPT(RAG) + AI뱅커 + 안랩 협력 인프라 위에 LLM 1단 추가. 6개월 PoC. |
| **혁신성(30)** | 28/30 | 글로벌·학술 단계, 금융 청약 적용은 세계 최초. GenAI 손실 시뮬레이션 영상 결합. |
| **합계** | **93/100** | |

### 2-6. AI 활용 포인트 (PT 발표 필수 포함)

1. **LLM 역질문 생성**: 상품 약관 RAG → 핵심 리스크 5개 추출 → 자연어 질문화 (우리은행 운영GPT 확장)
2. **의미 일치도 평가**: 고객 답변 임베딩 ↔ 정답 임베딩 cosine similarity + LLM judge
3. **적합성 자동 평가**: 답변 품질 + KYC + 거래 패턴 → 적합성 점수 (XAI로 직원·고객에 설명)
4. **GenAI 손실 시뮬레이션 영상**: Sora 류 video diffusion + 본인 자산·연령 입력 → 30초 개인화 시각화
5. **온디바이스 STT**: 음성 답변 → 텍스트 (우리은행 안랩 V3 협력 인프라)
6. **법적 디지털 증빙**: 영업점 녹취와 동일 효력의 청약 디지털 패키지 자동 생성

## 5. **왜 우리가 이걸 5일간 만들어야 하는가** — 정당성 설득 5문장

1. **법적 임박**: 2026.1.1 금소법 시행령 개정 + 보이스피싱 무과실 배상제(2,800억/년 부담) 하반기 입법 예고 → 사전예방 솔루션 부재 시 은행 손실 직접 발생.

2. **시장의 빈자리**: 9개 은행 + 정부 ASAP + 글로벌 + 핀테크 공모전 전수조사 결과, "고객 이해도 LLM 검증"은 **세계 어디에도 없음**. 1년 안에 누군가 만들 자리에 우리가 먼저.

3. **우리은행의 정체성과 일치**: 2026 "사전예방적 소비자보호 체계 전면 전환" + "생성형 AI FDS 고도화" 선언 + ELS 5개에서 제외된 보수적 판매 DNA = TalkBack AI가 회사가 가려는 방향을 정확히 가속화.

4. **ROI 정량 증명 가능**: ELS 1.4조 과징금 / 무과실 배상 2,800억 vs 시스템 구축 30~50억 = **100배 ROI**. 심사위원이 효과성 40점에 만점 주기 쉬운 구조.

5. **PT 데모 가능성**: 5분 안에 심사위원 앞에서 실시간 시연 가능. "당신이 이해했는지 묻습니다"라는 메시지는 한 줄로 박힘. 다른 팀은 도면·시퀀스 다이어그램만 보여줄 때 우리는 라이브 데모로 압도.
