# 브레인스토밍 — "이해 검증" 방향은 맞지만, 말 시키지 말자

> **작성일:** 2026-05-20
> **트리거:** 사용자 지적 — "말 못 하는 곳에선? 노인·장애인·바쁜 사람은? 인지부하 줄이는 게 맞지 않나?"
> **목표:** "진짜 이해"를 보장하면서도 **손님이 노력하지 않아도 되는** 방법 찾기

---

## 1. 왜 9개 은행 누구도 "이해 검증"을 안 했을까 — 5가지 진짜 이유

| 이유 | 내용 | 우리가 풀어야 할 것 |
|---|---|---|
| **① 인지부하** | 청약 막판에 손님한테 "다시 설명해보세요" 하면 짜증. 노인·장애인은 더 막힘 | 손님이 **노력 안 해도** 이해 측정해야 함 |
| **② 환경 제약** | 카페·지하철·사무실·새벽. 말 못 하는 곳이 절반 이상 | **말 안 시키는** 방법 필요 |
| **③ 이탈률 ↑ → 매출 ↓** | 청약 중간에 막히면 손님 도망. 영업 KPI와 정면충돌 | 청약 흐름을 **끊지 않는** 설계 |
| **④ 법적 부메랑** | "이해 확인했는데 손님이 이해 못 했다"는 기록이 오히려 은행 책임 강화 | **이해 못한 손님은 청약 자체를 막아야** 안전 |
| **⑤ 직원 부담** | 영업점에서 시간 2~3배 → 창구 적체 | **직원 개입 줄이는** 자동화 |

→ 그래서 다들 "약관 동의 체크박스 한 번"으로 끝낸 것. **편한 길**을 택한 결과가 ELS 1조 4천억.

## 2. 새로운 설계 원칙 4가지

1. **"손님은 가만히, AI가 일한다"** — 능동적 발화·답변 최소화
2. **"수동 신호로 이해도를 추정"** — 시선·머무름·스크롤·재생속도·표정
3. **"위험할 때만 끼어든다"** — 평소엔 자연스러운 청약 흐름 유지
4. **"끼어들 땐 한 화면에서 끝낸다"** — 신호등 한 번, 탭 한 번

---

## 3. 브레인스토밍 — 10가지 대안 (TalkBack 이름은 버림)

### A. 손님이 "보기만" 해도 되는 방향

**A-1. 시선·머무름 기반 수동 이해도 측정 (Passive Comprehension)**
- 핸드폰 전면 카메라로 시선 추적 (옵션, 동의)
- 핵심 위험 문단에 0.3초 미만 머무름 → 이해 안 됨
- 스크롤 속도 + 영상 빨리감기 + 일시정지 패턴
- → 손님은 그냥 평소대로 청약, AI가 뒤에서 평가

**A-2. AI 신호등 한 줄 평가**
- 손님 프로필(나이·자산·소득·기존보유)을 AI가 미리 알고 있음
- 청약 직전 화면에 단 한 줄: 🟢🟡🔴 + "당신 자산의 32%, 부담 큼"
- 손님은 보기만 하면 됨 (말 X, 답 X, 노력 X)

**A-3. AI 변호인(Devil's Advocate) — "사지 말라"고 먼저 말함**
- AI가 손님 입장에서 적극적으로 말림: "이 상품 사지 마세요. 이유 3가지"
- 손님이 그래도 사고 싶으면 "그래도 산다" 한 번 누름
- 인지부하 → AI가 대신 짊어짐

### B. 환경·취약계층 친화 방향

**B-1. 멀티모달 입력 — 음성/탭/그림/침묵 자동 전환**
- 폰 마이크가 주변 소음 감지 → 조용하면 음성, 시끄러우면 탭, 노인이면 큰 그림
- 응답 형식을 **환경에 맞춰 AI가 알아서 바꿈**
- 시각장애인 → 음성 / 청각장애인 → 진동·시각 / 발달장애 → 그림카드

**B-2. 24시간 쿨링오프 + 매일 한 줄 푸시**
- 청약 → 24시간 강제 대기 (인지부하 ZERO, 그냥 기다림)
- 그 24시간 동안 AI가 매일 짧은 푸시: "오늘은 이 시나리오, 손실 -28%"
- 24시간 뒤 손님이 "그래도 한다" 누르면 진행
- **법적으로도 청약철회권(7~14일)을 사전에 끌어온 효과**

**B-3. "30초 짧은 만화 → 탭 한 번"**
- 손실 시나리오를 30초 GenAI 웹툰으로 (텍스트 X)
- 손님은 다 본 뒤 🟢(이해함) 🔴(못 알겠음) 둘 중 하나 탭
- 🔴 누르면 다른 형식(영상·음성·그림·텍스트) 자동 전환

### C. "이해" 검증을 다르게 정의

**C-1. 평소 누적 학습 점수 (청약 시점엔 측정만)**
- 평소 우리WON뱅킹 쓸 때 게임·퀴즈·뉴스 카드로 금융 지식 누적
- 청약 시점에 누적 점수가 낮으면 → 보호 모드 자동 가입
- 청약 순간엔 아무것도 안 시킴 (인지부하 ZERO)

**C-2. AI 비서가 손님 대신 약관 읽음**
- 손님은 약관 안 봐도 됨. AI가 다 읽고 손님 프로필 기반 3줄 요약: "당신한테 위험한 부분 3가지"
- 손님은 3줄만 보면 됨 (인지부하 1/100)

**C-3. 가족·신뢰인 자동 동의 모드 (75세+ 또는 자발 등록)**
- AI가 "위험 청약"으로 판정하면 사전 등록한 자녀에게 즉시 푸시
- 자녀가 "엄마 이거 사도 돼" 누르기 전엔 보류
- 손님 본인은 노력 ZERO

### D. 게이미피케이션·체험형

**D-1. 본인 자산 시뮬레이터 — 슬라이더 한 번**
- "주가가 떨어진다면?" 슬라이더를 손가락으로 드래그하면 본인 자산이 실시간 변동
- 빨간 영역에 들어가면 폰이 진동
- 손님은 슬라이더 한 번만 만지면 됨

**D-2. 페르소나형 — "당신과 같은 30대 김민준씨"**
- AI가 손님과 같은 프로필 가상 인물의 5년 후 결과 보여줌
- "김민준씨는 -23%, 김민준씨처럼 되실 수 있어요"
- 보기만 하면 됨

## 4. 후보 묶음 3가지 — 골라야 할 방향

### 🅰 묶음 1. **"손님은 가만, AI가 다 한다" (Zero-Effort 보호)**
조합: A-2(신호등) + A-3(AI 변호인) + C-2(AI가 약관 대독)
- **핵심 메시지:** "약관 안 읽으셔도 돼요. AI가 다 읽고 당신한테 위험한 것만 알려드려요."
- **인지부하:** ★☆☆☆☆ (탭 1번)
- **취약계층:** ◎ (말 X, 읽기 X)
- **빈자리:** 글로벌·국내 모두 없음. AI 비서가 손님 편이라는 컨셉 자체가 신선
- **PT 데모:** "AI 변호인이 ELS 사지 말라고 말리는 30초 시연"

### 🅱 묶음 2. **"24시간 쿨링오프 + 매일 한 줄 푸시" (Slow Banking)**
조합: B-2 + A-2 + B-3(만화)
- **핵심 메시지:** "사고 싶을 땐 못 사요. 하루만 기다리세요. 그동안 우리가 매일 알려드릴게요."
- **인지부하:** ★☆☆☆☆ (대기만)
- **법적 의미:** 청약철회권을 사전화. 위법계약해지 방지
- **빈자리:** 청약 사전 쿨링오프 의무화는 글로벌 없음
- **PT 데모:** "어제 살 뻔한 ELS, 오늘 손님이 안 사기로 함" 시나리오

### 🅲 묶음 3. **"멀티모달 + 취약계층 자동 전환" (Adaptive Inclusion)**
조합: B-1(환경 감지) + B-3(만화/영상/음성/탭) + C-3(가족 모드)
- **핵심 메시지:** "노인도, 장애인도, 카페에서도, 새벽에도. 손님이 가능한 방식으로만 묻습니다."
- **인지부하:** ★★☆☆☆ (한 번은 답함, 단 형식 자유)
- **취약계층:** ◎◎◎ (응모분야 1 직격)
- **빈자리:** 단일 멀티모달 청약 시스템 없음. 신한 디지털데스크 10개 언어는 외국인 한정
- **PT 데모:** 카페 소음 → 자동 탭 모드 / 시각장애 모드 / 발달장애 그림카드 모드 시연

## 6. 강력 추천 — **🅰 + 🅲 결합**

> **"보호자 AI(Guardian AI)" — 손님 대신 AI가 약관 읽고, 손님 환경·상태에 맞춰서, 위험할 때만 한 줄로 알려준다.**

**한 화면 시나리오 (할머니):**
1. 손님이 ELS 청약 화면 진입
2. 폰 전면 카메라가 "노인 + 글씨 작게 봄" 감지 → 자동으로 큰 글씨 + 음성 모드
3. AI 변호인이 음성으로: **"이 상품, 어머님 자산의 40%예요. 손해 보면 한 달 생활비 절반이 사라져요. 사지 마시는 게 좋겠어요."**
4. 동시에 사전 등록된 따님 폰에 푸시: "어머님이 ELS 4천만원 청약 중. 확인 부탁드려요"
5. 할머니가 "그래도 하고 싶다" 누르면 24시간 대기 (B-2)
6. 24시간 동안 매일 한 줄 푸시. 따님과 통화 가능

**왜 1등 가능:**
- **인지부하 거의 0** — 할머니가 한 말씀도 안 함
- **취약계층 직격** (응모분야 1)
- **불완전판매도 동시 해결** (응모분야 2)
- **PT 데모 강력** — 할머니 시나리오 시연하면 심사위원 눈물
- **글로벌 빈자리** — 누적 학습 + 환경감지 + AI 변호인 + 쿨링오프 + 가족모드를 한 패키지로 묶은 곳 없음
- **우리은행 자산 100% 활용** — 시니어플러스 영업점 + AI-스미싱 안심(온디바이스) + 운영GPT(RAG) + AI뱅커
