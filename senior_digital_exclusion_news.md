## 1. 은행 점포 폐쇄 → 노인 금융 사각지대

### 수치
- 점포 수 변화: **2020년 말 6,427개 → 2025년 9월 5,523개 (904개·14.1% 감소)**
- **2025년 9개월 만에 전년도 1년 폐쇄량 초과** — 가속화 중
- 신규 규제 (2026.02): **반경 1km 통폐합도 주민 설문 의무화**

### 시사점
- 노년층 요구는 "단순 계좌"가 아니라 **연금 수령·병원비 이체·정기지출 설정** 같은 일상 밀착형
- 점포 사라지면 **금융 접근권 자체가 차단**되는 임계치
- 거동 불편 장애인·임산부에게도 동일한 문제

## 2. 디지털 격차 — 60대 vs 70대의 "절벽"

KISDI(정보통신정책연구원) 2024 조사: 60대는 '디지털 시니어'로 진화, **70대 이상은 완전히 다른 세대**.

### 스마트기기 활용능력 점수 (5점 만점)
| 분야 | 60대 | 70대 이상 | 격차 |
|---|---|---|---|
| 문자메시지·메신저 | 4.23점 (국민 평균 4.38) | 1점대 | 3점+ |
| 인터넷 | 3.39점 | **1.85점** | 1.54점 |
| 금융/쇼핑 등 | (상승 중) | 1점대 | 매우 큼 |

### 핵심 인사이트
- **70대 이상은 카톡은 되지만 은행앱은 못 씀** — "반쪽짜리 디지털 라이프"
- 65세 이상 노년층 중 **38%가 "은행앱 쓰고 싶다"고 응답** — **욕구는 있지만 진입을 못 함**
- 영국 사례: 상원에서 "디지털 포용 허브 + 오프라인 대안 의무화" 대응 채택

## 4. 진입 장벽 — 가입·인증 자체가 함정

### 4-1. OTP·공동인증서 무한루프
> "공동인증서 문제 해결하려면 OTP가 필요한데, OTP 등록하려면 공동인증서 인증이 필요함."

**해결책**: 신분증 + OTP 들고 **은행 점포 직접 방문**(그런데 점포는 사라지는 중)

### 4-2. 본인인증 실패의 구체적 원인
- **명의 불일치** / 주민등록정보 오류
- 문자 인증번호 미수신 (스팸·통신사 차단·저장공간 부족)
- **셀카 인증 실패** — 작은 화면, 짧은 제한시간, "표정 변화 요구" 등에서 좌절
- 정부24 본인인증조차 반복 실패 사례 다수
- 과기정통부 디지털정보격차 보고서: **고령층 디지털 역량 수준 54.5%** (정보취약계층 중 최저)

### 4-3. 비대면 계좌개설 — '한도제한계좌' 함정
- 보안 사고 예방을 위해 비대면 개설 계좌는 **이체 한도가 낮은 한도제한계좌**로 시작
- 일반 계좌 전환하려면 **앱으로 증빙 서류 제출** (또 한 번 디지털 진입 필요)
- 신분증 사진 보정·얼굴 변형으로 인식 오류 빈번
- 안심차단 서비스 활성화 시 개설 차단

### 4-4. 가입 단계 분해 — KB·토스 신규 가입 사례

**KB스타뱅킹 (기존 계좌 보유자)**
```
앱 설치 → 기본정보 입력 → 문자 인증 → 인증수단 선택 →
신분증 촬영 → 얼굴인증 → KB인증서 발급
→ 추가 계좌 없으면 ATM 직접 방문 (1원 송금 확인)
```

**토스 (신규 가입자)** — 11단계
```
문자 인증 → 기본정보 입력 → 비밀번호 설정 → 얼굴인식 허용 →
약관 동의(반복) → 통장 개설 → 신분증 촬영 →
얼굴인증(움직임 조작 필요) → 추가정보 입력 →
휴대폰 인증 → 토스 인증서 발급
```

### 4-5. "쉬운 모드"는 도달해야 쓸 수 있다
- 고령자 모드는 **앱 설치 + 본인인증 + 로그인 통과 후에만** 선택 가능
- **진입 자체가 막히면 쉬운 모드는 평생 못 봄**
- 기능 단순화가 아니라 **초기 진입 구조 자체의 재설계**가 필요

## 5. 사용 중 장벽 — 토스가 직접 조사한 시니어 UX 5가지

토스가 시니어 실사용자 인터뷰로 도출한 **5대 오인지 패턴** (toss.tech 공식):

### 5-1. 컴포넌트 오인지
- 텍스트·화살표(>)가 클릭 가능함을 인지 못 함
- 명확한 버튼 형태 아니면 **주변의 다른 진입점을 잘못 클릭**

### 5-2. 예시 이미지 오인지
- 입출금 내역 **예시 이미지의 '예시' 태그를 놓치고 본인 데이터로 오해**
- "내 이름이 나오니까 진짜인 줄 알았다"

### 5-3. 모션 그래픽 오인지
- 얼굴 등록 인트로 캐릭터 모션을 **100% 똑같이 따라해야 성공한다고 착각**

### 5-4. 라이팅 오인지
- "어떤 서류가 필요한가요?" → **앱이 자신에게 묻는 질문으로 문자 그대로 해석**
- 해결: "필요한 서류 확인하기" 명령형으로 바꿔야

### 5-5. 화면 구조 오인지
- 스크롤 가능 여부를 직관적으로 못 파악 → **눈에 보이는 영역만 읽고 끝**

### 시사점
- 시니어는 **추론을 안 함** — 명시되지 않은 것은 존재하지 않음
- WON AI 뱅커가 음성으로 "지금 화면 아래로 더 있어요"라고 알리면 5-5번 해결

## 7. 보이스피싱 & AI 딥페이크 — 고령층 폭격

### 수치
| 지표 | 수치 |
|---|---|
| 60대 이상 피해 건수 | **2016년 1,261건 → 2025년 5,801건 (4.6배 폭증)** |
| 60대 이상 비중 추이 | 2020년 16% → 2022년 20% → 2024년 25% → **2025 상반기 30.6%** |
| 전체 피해액 추이 | 2023년 4,472억 → 2024년 8,545억 → **2025년 8월 8,856억** (역대 최고) |
| 전체 발생 건수 | 2020년 3만 → 2024년 2만 (감소)지만 60대 비중은 증가 |

### 신종 AI 사기 (2025~2026)
- **30초 음성 데이터만으로** 가족 목소리 완벽 복제
- 영상통화에 얼굴까지 합성 — 2025 서울 직장인 '아들 얼굴' 딥페이크에 **5,000만원 송금**
- 시니어는 **"딥페이크"라는 개념 자체를 모름**
- 예방 권장: 가족만의 인증 키워드 (AI 모방 불가)

### 차단 인프라
- **한도제한계좌** (2016 도입): 비대면 개설 계좌 일일 한도 — **2025년 5월부터 500만원으로 상향**
- **계좌 지급정지**: 신고 시 30분~1시간 내 차단 가능
- **금감원 1332 / 경찰 112**: 즉시 신고 채널
